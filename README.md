# Chá Rifa 🍼

Site criado para ajudar uma amiga muito querida a arrecadar parte dos custos
da chegada do bebê dela, através de uma rifa online simples e acessível
pelo celular.

🔗 *Site:*https://kathleencris.github.io/cha-rifa/

## O desafio

Organizar a rifa de forma manual (papel ou WhatsApp) tornaria difícil
acompanhar quem já havia participado.

## A solução

Um site onde os convidados escolhem um número disponível em uma grade,
preenchem um formulário rápido e são direcionados ao WhatsApp para
confirmar o pagamento via Pix — tudo direto do celular, sem instalar nada.

## Resultado

38 participantes e R$ 760 arrecadados para ajudar na chegada do bebê. 💛

## Tecnologias

- HTML5
- CSS3
- JavaScript (armazenamento local dos números escolhidos)
